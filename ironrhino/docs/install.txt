1.install mysql,add two line after [mysqld] in my.conf or my.ini

character-set-server = utf8
collation-server = utf8_general_ci

if your mysql username is not "root" or password is not empty,please config jdbc.username and jdbc.password in applicationContext.properties
the default database is "ironrhino" and it will auto create if not exists,if you want to use another database,please config jdbc.url in applicationContext.properties

2.deploy /webapp to a server(tomcat,jetty,resin,glassfish ...)


3.open http://localhost:8080/setup in a browser(chrome,firefox,opera,IE7+ ...)
